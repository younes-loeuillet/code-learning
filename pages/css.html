Comprendre le CSS pour styliser vos pages web facilement

Le CSS est le langage qui permet de styliser et mettre en forme les pages web.

Qu'est-ce que le CSS ?

CSS signifie Cascading Style Sheets. C’est ce qui permet de définir l’apparence visuelle d’une page web.

Structure de base d'un fichier CSS

/* Sélecteur */
élément {
  propriété: valeur;
}

Un fichier CSS contient des règles qui s’appliquent à des éléments HTML pour modifier leur style (couleur, taille, marges, etc.).

Les propriétés CSS les plus utilisées

  • color : définit la couleur du texte.
  • background-color : définit la couleur de fond.
  • font-size : contrôle la taille du texte.
  • margin : espace autour des éléments.
  • padding : espace à l'intérieur des éléments.
  • border : ajoute une bordure autour d’un élément.
  • display : contrôle comment un élément est affiché (block, inline, flex...).
  • flex : utilisée pour gérer des mises en page flexibles.
  • position : contrôle la position des éléments (relative, absolute, fixed).

À retenir

  • Le CSS permet de contrôler l’apparence des pages web.
  • Les règles CSS ciblent des éléments HTML pour leur appliquer des styles.
  • Le CSS est essentiel pour rendre les sites attrayants et lisibles.
