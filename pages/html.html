Apprendre le HTML

Le HTML est le langage de base pour créer des pages web. Il structure le contenu d'un site.

Qu'est-ce que HTML ?

HTML signifie HyperText Markup Language. C’est le squelette de toute page web.

Structure de base d'une page

<!DOCTYPE html>
<html>
  <head>
    <title>Titre de la page</title>
  </head>
  <body>
    Contenu visible
  </body>
</html>

Les balises les plus utilisées

  • <h1> à <h6> : Ces balises définissent les titres sur une page HTML. <h1> est le titre principal, le plus important, tandis que <h6> est un titre très secondaire. Ils structurent le contenu et aident aussi au référencement.
  • <p> : Sert à créer un paragraphe de texte. C’est la balise de base pour écrire des phrases et des blocs de texte dans une page.
  • <img> : Permet d’insérer une image dans la page. Elle utilise l’attribut src pour indiquer le chemin de l’image et alt pour une description textuelle si l’image ne s’affiche pas.
  • <a href=""> : Crée un lien cliquable. L’attribut href contient l’URL ou le chemin vers la page cible. En ajoutant target="_blank", le lien s’ouvre dans un nouvel onglet.
  • <ul>, <ol>, <li> : Ce sont les balises pour créer des listes. <ul> crée une liste à puces, <ol> une liste numérotée, et <li> désigne chaque élément de la liste.
  • <div> : Un conteneur général qui sert à regrouper plusieurs éléments ensemble pour appliquer un style ou organiser la mise en page. Il n’affecte pas directement le contenu.
  • <span> : Balise en ligne utilisée pour cibler une partie spécifique de texte à l’intérieur d’un paragraphe ou d’une phrase. Utile pour appliquer un style ou un script sans casser la structure.
  • <br> : Insère un saut de ligne (retour à la ligne) sans créer un nouveau paragraphe. Utile pour des textes courts ou des adresses.
  • <strong> et <em> : <strong> met en avant un texte important (souvent affiché en gras), tandis que <em> met le texte en emphase (souvent en italique). Ce sont aussi des balises sémantiques, utiles pour l’accessibilité.
